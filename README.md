<h1 align="center">KanaNya | 日语假名练习</h1> 

KanaNya 是一个为日语初学者准备的假名练习小工具，通过对平假名、片假名和罗马音选择配对来强化你的记忆。

GitHub Pages:
https://leafnya.github.io/japanese-kana-practice
